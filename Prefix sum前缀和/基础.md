# Prefix Sum

### Purpose:

在多次区间求和的场景下，把「每次从头加」的 **O(n)** 计算，

变成一次预处理 + 每次 **O(1)** 查询。

### 算法模板

```cpp
    vector <long long> v(n + 1, 0), prefixsum(n + 1, 0);

    //v和它的前缀和prefixsum
    for (long long i = 1; i <= n; i++) {
        cin >> v[i];
        prefixsum[i] = prefixsum[i - 1] + v[i];
    }

    //使用前缀和prefixsum处理区间[a, b]的累加
    while (q--) {
        long long a, b;
        cin >> a >> b;
        long long sum = prefixsum[b] - prefixsum[a - 1];        
        cout << sum << '\n';
    }
```

## 构造字符串

给定三个正整数 n、a 和 b。你需要构造一个长度为 n 的字符串 s，由小写拉丁字母组成，使得 **每个长度为 a 的子串都恰好包含 b 个不同的字母**。题目保证答案一定存在。

你需要回答 t 组相互独立的测试用例。

我们回顾一下，子串 s[l…r] 指的是字符串 sl, sl+1, …, sr，它的长度为 r−l+1。在这个问题中，你只需要关注长度为 a 的子串。

**输入**

第一行输入一个整数 t (1 ≤ t ≤ 2000) —— 测试用例数量。接下来有 t 行，每行包含三个整数 n、a 和 b (1 ≤ a ≤ n ≤ 2000, 1 ≤ b ≤ min(26, a))，其中 n 是需要构造的字符串长度，a 是子串的长度，b 是每个长度为 a 的子串中要求的不同字母数量。

保证所有测试用例中 n 的总和不超过 2000（∑n ≤ 2000）。

**输出**

对于每个测试用例，输出任意一个满足条件的字符串 s，长度为 n，且每个长度为 a 的子串都恰好有 b 个不同字母。若答案不唯一，可输出任意一个。题目保证至少有一个解。

**示例**

输入：

```
4
7 5 3
6 1 1
6 6 1
5 2 2
```

输出（可能不同，仅为示例）：

```
tleelte
qwerty
vvvvvv
abcde
```

**说明**

示例 1 的所有长度为 5 的子串：

- "tleel": 含 3 个不同字母
- "leelt": 含 3 个不同字母
- "eelte": 含 3 个不同字母





## 解题思路

题目要求：

- 每个长度为 **a** 的子串里有 **恰好 b 个不同字母**

最稳的构造方法是：

#### 用前 b 个字母循环即可
 比如 b = 3，就循环 `"abcabcabc..."`
 比如 b = 1，就循环 `"aaaaaaa..."`
 比如 b = 5，就循环 `"abcdeabcde..."`

 ❓ **为什么只循环 b 个字母就满足“每个长度为 a 的子串都有 b 个不同字母”？**

因为：

- 你构造的是无限循环 `"abc... (b 个字母)"` 的周期串
- 不管窗口 a 多大，它跨过多个周期，但周期里本来就只有这 b 个字母
- **窗口中“不可能出现超出这 b 个字母”**
- **窗口又足够长，一定会覆盖到所有 b 个字母**
- 因此窗口中不同字母永远是恰好 b 个

（题目保证答案存在，说明这种构造是合法的）

## 代码实现

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int t;
    cin >> t;

    while (t--) {
        int n, a, b;
        cin >> n >> a >> b;

        // 预分配空间，避免多次扩容，提高性能
        string s;
        s.reserve(n);

        // 关键构造：循环使用前 b 个字母（a 与构造无关）
        // i % b 会让字母按 "abc... (b个)" 的顺序重复出现
        for (int i = 0; i < n; i++) {
            s += char('a' + (i % b));   // 'a'+0='a', 'a'+1='b',...
        }

        // 输出结果
        cout << s << "\n";
    }

    return 0;
}
```

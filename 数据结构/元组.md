# 一般配合STL容器食用

# pair

pair用来**把两个数据打包在一起**当作一个整体使用。（双元组）

```cpp
#include <utility>
using namespace std;

//初始化
pair<类型1, 类型2> 变量名;
pair<int, string> p = {1, "apple"};
pair<int, int> a = {1, 2}, b = {1, 3};  //定义a和b两个pair
pair<int, int> a(1, 2), b(1, 3);

//访问元素
cout << p.first;   // 输出 1
cout << p.second;  // 输出 "apple"

//修改元素
p.first = 2;
p.second = "banana";

//与stl容器使用

    }
```

# tuple

tuple 是一个可以同时存储多个不同类型元素的固定长度容器。（多元组）

```cpp
#include <tuple>
using namespace std;

//初始化
tuple<int, string, double> t = {1, "apple", 3.14};

//访问元素
cout << get<0>(t);  // 输出 1
cout << get<1>(t);  // 输出 "apple"

//修改元素
get<2>(t) = 2.71;  // 修改第三个元素
```

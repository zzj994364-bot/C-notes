### 标准库实现形式

```cpp
#include <iostream>
#include <algorithm>

auto it = lower_bound(v.begin(), v.end(), x);
```

| 函数名                                  | 查找条件 | 含义          |
| ------------------------------------ | ---- | ----------- |
| lower_bound(v.begin(), v.end(), x)   | >= x | 第一个大于等于     |
| upper_bound(v.begin(), v.end(), x)   | > x  | 第一个大于       |
| binary_search(v.begin(), v.end(), x) | 精确匹配 | 返回(1或0）是否存在 |

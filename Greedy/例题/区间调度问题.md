## Movie Festival

在一个电影节上会播放 n 部电影。你知道每部电影的开始时间和结束时间。你最多能够完整地看多少部电影？

**输入**
 第一行输入一个整数 n：电影的数量。
 接下来有 n 行，每行包含两个整数 a 和 b：表示一部电影的开始时间和结束时间。

**输出**
 输出一个整数：你最多能看的电影数量。

**限制条件**
 1 ≤ n ≤ 2·10⁵
 1 ≤ a < b ≤ 10⁹

**示例**
 输入：
 3
 3 5
 4 9
 5 8

输出：
 2









## 解题思路

按结束时间排序，按最早开始的电影开始选

> 选择当前能看的电影中 **结束时间最早的那一部**，
>  能给未来留出最多的时间，使可看的总数量最大。

这是区间调度的经典可证明贪心。





## 代码实现

```cpp
#include <iostream>
#include <vector>

struct movietime {
    int start;
    int end;
};

int main () {
    int n;
    cin >> n;
    vector<movietime> a(n);
    for (int i = 0; i < n; i++) cin >> a[i].start >> a[i].end;
    
    sort(a.begin(), a.end(), [](movietime x, movietime y) {
        return x.end < y.end;
    });
    
    int last_end;
    int cnt;
    for (auto x : a) {
        if (x.start >= last_end) {
            cnt++;
            last_end = x.end;
        }
    }
    
    cout << cnt << endl;
    
    return 0;
}
```

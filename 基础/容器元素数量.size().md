# .size()

## 1. 基本作用

.size() 是 容器（如 string、vector、set、map 等）的成员函数，用于返回容器中**元素的数量**。不可以用给int。

## 2. 常见用法示例

- 字符串长度：

```cpp
string s = "hello";
cout << s.size();  // 输出 5
```

- 向量（vector）元素数量：

```cpp
vector<int> v = {1, 2, 3};
cout << v.size();  // 输出 3
```

- 集合（set）中元素数量：

```cpp
set<char> st = {'a', 'b', 'a'};
cout << st.size();  // 输出 2，重复元素只计一次
```

- 判断一个整数是几位数

```cpp
int n = 12345;
string s = to_string(n);
cout << s.size() << endl;  // 输出 5（说明是五位数）
```

- 映射（map）中键值对数量：

```cpp
map<int, string> m;
m[1] = "apple";
m[2] = "banana";
cout << m.size();  // 输出 2
```

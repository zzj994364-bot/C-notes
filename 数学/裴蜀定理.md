### 📘 裴蜀定理（Bézout's Identity）

#### 🧩 定理内容：

对于任意两个整数 a 和 b（不全为零），存在整数 x 和 y，使得：a⋅x  +  b⋅y  =  gcd⁡(a,b)

这称为**裴蜀等式（Bézout equation）**。

#### 📌 推论：

如果 d = gcd(a, b)，那么对任意整数 c，

方程：a⋅x  +  b⋅y  =  c

有整数解 **当且仅当** d 能整除 c，即：c  mod  d  =  0  ⇔  d  |  c

#### ✏️ 举例说明：

- 例 1：
  a=6,      b=9,      gcd⁡(6,9)  =  3
  ⇒ 有整数 x = -1, y = 1 满足：6⋅(−1)+9⋅1=−6+9=3

- 例 2：
  想解 6x+9y=12，因为 12%3=0，有解
  实际解是裴蜀等式的解乘上 4：x=−4,y=4 ⇒ 6⋅(−4)+9⋅4=12

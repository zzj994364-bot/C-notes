## 代码模板
```cpp
// 快速幂函数
long long power(long long base, long long exp, long long mod) {
    long long result = 1;
    base %= mod;
    while (exp > 0) {
        if (exp & 1) {
            result = (result * base) % mod;
        }
        exp >>= 1;
        base = (base * base) % mod;
    }
    return result;
}
```



## 函数作用：计算

```
(base^exp) mod mod
```

------

### 🔹 1. base 是什么？

**base = 底数**

在表达式 `a^b mod m` 中的：

- `a` 就是 **base**

例子：

```
3^13 mod 1000000007
base = 3
```

------

### 🔹 2. exp 是什么？

**exp = 指数 exponent**

在 `a^b mod m` 中的：

- `b` 就是 **exp**

例子：

```
3^13 mod 1000000007
exp = 13
```

------

### 🔹 3. mod 是什么？

**mod = 取模的模数**

在 `a^b mod m` 中的：

- `m` 就是 **mod**

常见模数：

- `1e9+7`
- `998244353`
- 或者任意你给的整数

例子：

```
3^13 mod 1000000007
mod = 1000000007
```

------

### 🔹 4. return result 是什么？

**它返回的就是：**

```
(base^exp) % mod
```

也就是 **幂运算之后再取模的结果**。

------

###  一个完整例子

调用：

```
power(3, 13, 1000000007);
```

函数返回：

```
1594323
```

因为：

```
3^13 = 1594323
1594323 mod 1000000007 = 1594323
```

------

###  再举一个大数的例子

```
power(2, 1000000000000LL, 1000000007);
```

函数返回的是：

```
2^1000000000000 mod 1000000007
```

这个值正常用计算机要做一万亿次乘法，而快速幂只需要 **log₂(1e12) ≈ 40 次循环**。

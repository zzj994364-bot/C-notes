#### **1. `cd`**

改变目录（换文件夹）

```
cd 文件夹名
cd ..
cd C:\path\to\folder
```

------

#### **2. `dir`**(directory)

查看当前目录下所有文件和文件夹
 就像打开抽屉往里看。

```
dir
```

------

#### **3. `cls`**

清屏
 把窗口刷干净，像擦白板。

```
cls
```

------

#### **4. `mkdir`**

创建一个新文件夹（make directory）

```
mkdir myfolder
```

------

#### **5. `del`**

删除文件（小心使用，不会放进回收站）

```
del test.cpp
```

------

#### **6. `copy`**

复制文件

```
copy a.txt b.txt
```

------

#### **7. `move`**

移动或重命名文件

```
move a.txt folder\
move oldname.cpp newname.cpp
```

------

#### **8. `type`**

查看文本文件内容（不用打开 Notepad）

```
type test.cpp
```

------

#### **9. `exit`**

退出终端

```
exit
```

------


#### **10. g++ 编译**

```
g++ test.cpp -o test
```

生成 test.exe。

------

#### **11. 运行可执行文件**

```
.\test
```
